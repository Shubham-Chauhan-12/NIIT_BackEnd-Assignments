Steps to create MicroService
1.Create MavenQuickStartProject
2.Remove src folder
3.copy and paste restful services
4.configure POM(1-parent POM/2-child POM)
4.a---Parent POM file--add Parent tag(with below 4 lines)

  <parent>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-parent</artifactId>
    <version>2.7.5</version>
    <relativePath/> <!-- lookup parent from repository -->
  </parent>

4.b---in Parent POM----add Common dependencies of child services inside Restfulservice project
spring-boot-web,spring-boot-test,jpa

4.c---in parent POM---add modules tag--specify child Restful Service
  <modules>
    <module>UserAuthenticationService</module>
    <module>UserMovieService</module>

  </modules>
4.d---in parent POM--add surefire plugin

<build>
  <plugins>
    <plugin>
      <groupId>org.apache.maven.plugins</groupId>
      <artifactId>maven-surefire-plugin</artifactId>
      <configuration>
        <useSystemClassLoader>false</useSystemClassLoader>
      </configuration>
    </plugin>
  </plugins>

  </build>



4.e add Packaging tag in Parent pom
  <packaging>pom</packaging>



5.Configure Child POM's

5.a--change Parent tag
	<parent>
		<groupId>org.springframework.boot</groupId>
		<artifactId>spring-boot-starter-parent</artifactId>
		<version>2.7.5</version>
		<relativePath/> <!-- lookup parent from repository -->
	</parent>

change above parent in child POM as
	<parent>
		 <groupId>com.niit</groupId>
 	 	<artifactId>MovieApplication</artifactId>
  		<version>1.0-SNAPSHOT</version>
	</parent>


5.b Remove commom dependencies like  spring-web,test and JPA
but have MYSQL or MONGO
